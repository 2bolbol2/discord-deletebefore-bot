# Discord Delete Bot v5 ETA

نسخة محسنة لإدارة حذف رسائل Discord من Termux.

## المميزات

- `/deletebefore` حذف كل الرسائل قبل رسالة معينة.
- `/deleteafter` حذف كل الرسائل بعد رسالة معينة.
- `/deleteuser` حذف رسائل عضو محدد في القناة.
- `/stopdelete` إيقاف العملية الحالية.
- رسالة عداد مؤقتة في القناة يتم تحديثها تلقائيًا كل 5 ثوانٍ.
- عرض النسبة المئوية و ETA بعد مرحلة حساب عدد الرسائل.
- حذف رسالة العداد تلقائيًا عند الانتهاء.
- رسالة انتهاء واضحة في Termux.

## التشغيل

```bash
cd ~/discord-deletebefore-bot
git pull
npm install
BOT_TOKEN="التوكن_الجديد" CLIENT_ID="Application_ID" GUILD_ID="Server_ID" npm start
```

## مهم

هذه النسخة تقوم بمرحلة حساب أولًا قبل الحذف حتى تعرض النسبة والوقت المتبقي بدقة. لذلك قد تبدأ أبطأ قليلًا، لكنها تعطي عدادًا أدق.
